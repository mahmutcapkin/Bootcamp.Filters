# Filters in .NET6(Web API)
## About Project
This is the project I realized to reinforce what I learned about middleware, 
filters and validation during the bootcamp process.

 ## Built With
Project is created with:
* .NET6(Web API)
* AutoMapper
* FluentValidation
##
Within the Project:
* Middleware
* Filters
  -ActionFilter
  -AuthorizationFilter
  -ResourceFilter
  -ExceptionFilter
* DTOs(Response)
